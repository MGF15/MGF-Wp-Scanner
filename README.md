MGF-WP-Scanner
==============
MGF-WP-Scanner is a simple WordPress scanner.


Requirements
---------------

* Python 2.7 (Obviously)
* colorama (https://pypi.python.org/pypi/colorama) # for terminal coloring
* requests (https://pypi.python.org/pypi/requests) # for security

# Help 

```$ python(2) MWS.py -h```
